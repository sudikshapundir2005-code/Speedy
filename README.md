### ⚠️ Note for Visitors
If you see a "Dangerous Site" warning when opening the website, please follow these steps:
1. Click on the **"Details"** button on the warning page.
2. Click on the link that says **"visit this unsafe site"** or **"proceed to site"**.

**Why does this happen?**
This warning appears because the website is hosted on a free domain and currently lacks an SSL certificate (HTTPS), which is common for new, free-hosted projects. It is safe to proceed!
